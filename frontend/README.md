# Task Manager Frontend

This is a React-based UI for your Spring Boot Task Manager backend.

## Setup

1. Open a terminal in the `frontend` folder.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the app (default: http://localhost:3000).

## Usage
- Add, edit, and delete tasks.
- All changes are synced with the backend at http://localhost:8080.

## Backend
Make sure your Spring Boot backend is running on port 8080.

## CORS
CORS is enabled for `http://localhost:3000` in the backend controller.
